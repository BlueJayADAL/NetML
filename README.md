# NetML
The NetML project using Intel DAAL

## Walkthrough Guide
1. It is recommended to have an Anaconda Distribution of Python installed, as in this guide Anaconda will be used to create a virtual environment to install Intel DAAL.
2. Install [Joy](https://github.com/cisco/joy) (with "-gzip-enabled") and run it successfully. Note that the current Joy cannot generate the DNS data correctly when the 'bidir' is enabled which is required by TLS data. In order to fix this issue, please overwrite the current dns.c of Joy with the one in the JoyFix.
